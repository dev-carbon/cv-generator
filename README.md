# CV Generator

You can access components and modify their content to personalize your CV.

## Available Components
- **Header**
- **Achievements**
- **AchievementCard**
- **Education**
- **EducationCard**
- **Experience**
- **ExperienceCard**
- **Skills**

Run ```npm run dev``` to see a preview.

Run ```npm run export``` to export your CV as a PDF.
